PintoFly

A custom Firefly client using JoshHeng's Firefly API

Another thing I'm working on is a custom Firefly client, which is intended to be as lightweight as possible whilst retaining all the key functionality of the normal website. If you didn't know, Firefly is an online learning platform, used by schools to send out and manage tasks. There is currently a few major issues with it, such as the fact that the task list on your 'dashboard' is in the wrong order, the 'mark all as done' function is really broken, and that the website is really slow and unoptimised. With this project, I aim to fix all of that, as well as adding some way of custom styling the UI using CSS.
